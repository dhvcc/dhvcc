Dwapper.tech

While being a pretty simple service, it was build with as much Django as I could just to get a taste of the framework

The site provides you with pretty nice security headers such as CSP, FP, CORS. Protection against CSRF and password bruteforce. It uses SSL encryption as well as it only supports TLS 1.2/1.3

Account and password reset are confirmed via e-mail. The API is built using Django's DRF. Though it lacks in template caching and i18n it would be a total overkill for a small project like this one
